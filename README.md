# Iron-Matrix4 Home Assistant Integrations

This repository contains custom integrations for Home Assistant.

## Available Integrations

### Jarvis AI Conversation Agent

Enable HTTP API integration for the [Jarvis AI Add-on](https://github.com/Iron-Matrix4/ha-addons).

[![Open your Home Assistant instance and start setting up a new integration.](https://my.home-assistant.io/badges/config_flow_start.svg)](https://my.home-assistant.io/redirect/config_flow_start?domain=jarvis_conversation)

### Installation (HACS)

1. Open HACS.
2. Add this repository (`https://github.com/Iron-Matrix4/ha-integrations`) as a **Custom Repository** (Category: Integration).
3. Install **Jarvis AI Conversation Agent**.
4. Restart Home Assistant.
